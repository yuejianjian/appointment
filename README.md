# appointment
基于react开发的学员预约系统

## 技术栈

react+react-router-dom + ES6/7 +antd+axios


## 项目运行


```
git clone https://github.com/yuejianjian/appointment.git  

cd appointment  

npm install 或 yarn(推荐)

npm start (访问线上后台系统)

访问: http://localhost:3000
